Hi this project will be for lrarning purpose 
of how a basic DE end-to-end pipeline looks like

Things i will try to implement in this project:
1. Data Ingestion
2. Data filtering
3. Data transformation
4. Data loading
5. Data validation

DataSet: Covid data set
Source: https://www.kaggle.com/datasets/imdevskp/corona-virus-report

Setup:
1. Create a virtual environment
    (python -m venv venv)- to create a virtual environment that will isolate the project dependencies from the global Python environment.
    (source venv/bin/activate)- to activate the virtual environment, 
    allowing you to work within the isolated environment and install packages without affecting the global Python installation.
    
2. Install the required libraries using pip install -r requirements.txt
